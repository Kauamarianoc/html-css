Texto deletado

Podemos marcar um texto com a tag DEL, para indicar que aquele texto deve ser lido mas não considerado.

Texto inserido

Podemos também marcar um texto com a tag INS, para indicar que aquele texto foi inserido para ser lido e dar ênfase

Podemos escrever texto sobrescrito usando a tag SUP, como exemplo x2+7
